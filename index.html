Examen du 10 mai 2017

Les objectifs de cet exercice:

  1. Clôner ce projet sur votre ordinateur avec Github Desktop.
  2. Une webfont est incluse dans ce projet, mais elle n'est pas chargée. Faire en sorte que cette fonte soit visible.
  3. La couleur de ce site est un peu trop terne et triste. Changer les couleurs du fond et du texte pour obtenir une atmosphère joviale et printanière.
  4. Des Media Queries figurent dans le code CSS, mais n'ont aucun effet. Faire en sorte que les @mediaqueries s'appliquent
  5. Après avoir effectué ces corrections, bien vérifier que vos commits aient été synchronisés sur Github.
